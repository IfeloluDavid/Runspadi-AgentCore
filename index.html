About Runspadi

Nigeria's last-mile delivery, fully automated

Runspadi connects restaurants, grocery stores, and e-commerce businesses with independent riders across Lagos, Abuja, and Port Harcourt. What used to require 15 dispatchers working phones and WhatsApp now runs on six coordinated AI agents — processing 800+ orders daily with zero manual intervention.

Built by Nerdshub on Amazon Bedrock AgentCore, the system validates addresses, assigns the nearest rider, sends real-time SMS updates, monitors every delivery, escalates edge cases, and generates daily analytics — all autonomously.

Order Reception Agent

Validates both pickup and delivery addresses via geocoding, checks restaurant availability in DynamoDB, and creates the delivery job. Escalates immediately if addresses are invalid or the restaurant is closed.

  • validate_address → Google Maps geocoding
  • check_restaurant_availability → DynamoDB
  • create_delivery_job → DynamoDB orders table
  • escalate_to_human → DynamoDB + SNS
